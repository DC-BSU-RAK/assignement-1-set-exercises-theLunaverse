HTML & CSS Quick Reference Guide

HTML Tags

<p>


<p>This is a paragraph</p>

Defines a paragraph of text

<h1> to <h6>


<h1>Main Heading</h1>
<h2>Subheading</h2>

Defines headings with different importance

<a>


<a href="https://www.example.com">Link Text</a>

Creates a hyperlink to another webpage

<img>


<img src="image.jpg" alt="Description">

Embeds an image on the webpage

<ul> and <li>


<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

Creates an unordered list

<ol> and <li>


<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

Creates an ordered list

<div>


<div>
    Container for other elements
</div>

Generic container for content

<u>


<p>This is <u>underlined</u> text</p>

Underlines text

<strong>


<p>This is <strong>important</strong> text</p>

Indicates text of strong importance

<small>


<p>This is <small>small</small> text</p>

Turns text to smaller font size

CSS Properties

color


p {
    color: blue;
}

Changes text color

background-color


div {
    background-color: yellow;
}

Sets background color

font-size


h1 {
    font-size: 24px;
}

Changes text size

margin


p {
    margin: 10px;
}

Adds space around an element

padding


div {
    padding: 15px;
}

Adds space inside an element

border


img {
    border: 2px solid black;
}

Adds a border around an element

text-align


h1 {
    text-align: center;
}

Aligns text

width


div {
    width: 300px;
}

Sets the width of an element
