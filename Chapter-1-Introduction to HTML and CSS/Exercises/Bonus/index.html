HTML & CSS Quick Reference Guide

HTML Tags

<p>


<p>This is a paragraph</p>

Defines a paragraph of text

<h1> to <h6>


<h1>Main Heading</h1>
<h2>Subheading</h2>

Defines headings with different importance

<a>


<a href="https://www.example.com">Link Text</a>

Creates a hyperlink to another webpage

<img>


<img src="image.jpg" alt="Description">

Embeds an image on the webpage

CSS Properties

color


p {
    color: blue;
}

Changes text color

background-color


div {
    background-color: yellow;
}

Sets background color
